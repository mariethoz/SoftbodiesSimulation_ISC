Detailed Description

A 2D vector class providing basic arithmetic, geometric operations, and JSON serialization.

This struct represents a mathematical vector in 2D space with components (x, y). It supports vector arithmetic, normalization, dot/cross products, and serialization to/from JSON for persistence.
