Detailed Description

GDSimulation is the Godot wrapper for the Simulation class.

Warning
This file is deprecated and will be removed in future versions. It will not be replaced. Please use the new GDSimulation_2 classes instead.

It extends Node2D to allow for 2D positioning within the Godot scene. This class manages the simulation lifecycle, including stepping the simulation and rendering the simulation state.
