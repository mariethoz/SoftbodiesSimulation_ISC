Detailed Description

A Resource representing a constraint between two particles in the simulation.

Warning
This file is deprecated and will be removed in future versions. It will not be replaced. Please use the new GDSoftBody_2 classes instead.

This resource defines a constraint that connects two particles, controlling their relative motion based on stiffness and damping parameters.
