Detailed Description

A Resource representing a particle in the soft body simulation.

Warning
This file is deprecated and will be removed in future versions. It will not be replaced. Please use the new GDSoftBody_2 classes instead.

This resource defines a particle with position, mass, radius, and pinning properties for use in soft body simulations.
