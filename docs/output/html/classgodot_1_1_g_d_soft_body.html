Detailed Description

A Resource representing a soft body in the simulation.

Warning
This file is deprecated and will be removed in future versions. It will not be replaced. Please use the new GDSoftBody_2 classes instead.

This resource defines a soft body composed of particles and constraints, along with physical properties such as friction, restitution, stiffness, and damping.
